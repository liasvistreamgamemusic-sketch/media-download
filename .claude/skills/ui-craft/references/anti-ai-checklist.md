# アンチ AI チェックリスト — 「AI が作った」と見えるパターンの排除

AI 生成 UI が似通う根本原因: モデルは学習データの「統計的中央値」(Tailwind 既定の indigo、
Linear 風ダークヒーロー等) に回帰する。だから**意識的に外さない限り、必ずこの見た目になる**。

使い方: Phase 1 (選択肢設計) で「入れない」ため、Phase 4 (納品前) で「残っていないか」のために
全項目を diff と突き合わせる。**1 つでも該当 = 直してから納品**。
(例外: ユーザーが明示的に指定したもの・プロジェクト既存の規約はこの限りでない)

## 色・エフェクト

- [ ] 紫/インディゴ→青のグラデーション (ボタン・テキスト・背景・光る球体・オーロラ背景)
- [ ] グラデーションテキストの見出し・数値
- [ ] 理由なきデフォルトダークモード + 色付き box-shadow の「グロー」、ダーク地にシアン
- [ ] 装飾目的のグラスモーフィズム (blur カード・光るボーダー) の乱用
- [ ] 「上品風」の反射的なクリーム/ベージュ背景 (過剰修正系の新しい定番)
- [ ] 同一カードにヘアラインボーダー + 大きい拡散影の両掛け
- [ ] 全要素一律 opacity 0.1 の影 / 全要素に角丸 / 小さい要素への 24px+ の過剰角丸

## タイポグラフィ

- [ ] Inter / Roboto / Arial / Geist / Space Grotesk / Instrument Serif をデフォルト採用
- [ ] 巨大イタリックセリフのヒーロー見出し・サンセリフ本文への「セリフイタリック 1 単語」挿し
- [ ] H1 の上のピル型バッジ (eyebrow)・全セクションに繰り返す小さな uppercase トラッキングラベル
- [ ] サイズ差の乏しいフラットな階層 (隣接ステップ比 1.25 未満)
- [ ] 本文へのワイドトラッキング (0.05em 超)

## レイアウト・コンポーネント

- [ ] **中央寄せヒーロー (見出し+CTA) → アイコン付きカード 3 枚グリッド**のテンプレ構成
- [ ] **角丸カードの色付き左ボーダー** (「単一で最も分かりやすい AI シグナル」とまで言われる)
- [ ] アイコンを小さい角丸タイルに入れて見出しの上に積むフィーチャーカード
- [ ] 同サイズカードの均一グリッドの連打・カードの中のカード (ネスト)
- [ ] 「01 / 02 / 03」のセクション番号・脈絡のない 3 ステップ構成
- [ ] 統計バナー行 (大きい数字 + 小さいラベル ×3)
- [ ] 絵文字をアイコン・箇条書き記号に使用
- [ ] 全箇所同一 gap の単調な余白 (密→疎のリズムがない)
- [ ] 行長 80ch 超の本文

## モーション・コピー

- [ ] 全部が動く: 跳ねるボタン・揺れるアイコン・浮遊するバッジ (意味のないモーション)
- [ ] インターフェース要素への bounce / elastic イージング
- [ ] ホバーでの画像 scale 拡大 (惰性でやらない)
- [ ] 「効率化を実現」「〜を加速」「世界クラス」等の空虚な定型句、em-dash 乱用

## 代わりにやること

| 排除したもの | 置き換え |
| --- | --- |
| 紫グラデ | 視点のあるパレット: 高コントラスト黒+鮮烈な 1 色 / 温かいアーストーン / 深緑・ペトロールブルー / クリーム+ピンク等。OKLCH でランプを組み、トークン表で固定 |
| Inter 一択 | 見出し=個性 (Bricolage Grotesque, Cabinet Grotesk, Clash Display, Playfair 等) + 本文=可読、の 2 書体構成 |
| カード 3 枚グリッド | **1 つの強いレイアウトプリミティブの反復** (非対称 2 カラム、罫線区切りのエディトリアル行、オーバーラップ等)。階層は余白・タイポ・罫線 1 本で |
| アイコンタイル | 文脈に合う図版・数字そのもの・写真・カスタム SVG。アイコンが必要なら裸で置く |
| 意味のないモーション | ロード時 1 回のオーケストレーション + 状態変化を伝えるトランジションのみ (motion.md) |
| 定型コピー | 具体的な名詞と数字。そのドメインの実物らしい文言 |

## 最後の問い (Phase 4 で必ず)

> このページからロゴを外しても「このプロダクトのページだ」と分かるか?
> 分からなければ、それはテンプレであってデザインではない。方向性から出直す。
